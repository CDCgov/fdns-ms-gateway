Welcome to Foundation Services

Foundation Services (FDNS) are a suite of microservices offering easy-to-use endpoints, specifically designed for public health software use cases.

FDNS HL7 Utilities Microservice

This is the microservice with the HL7 utilities service to parse, validate and generate sample HL7 data.

http://origin/hl7/

FDNS CDA Utilities Microservice

This is the microservice with the CDA utilities service to parse, validate and generate sample CDA data.

http://origin/cda/

FDNS Storage Microservice

This microservice contains the Storage layer for the Data Lake. This is the immutable layer.

http://origin/storage/

FDNS Object Microservice

This microservice contains the Object layer for the Data Lake. This is the mutable layer of the data lake.

http://origin/object/

FDNS Indexing Microservice

This microservice contains the Indexing layer for the Data Lake. This is the navigation layer.

http://origin/indexing/

FDNS Combiner Microservice

This is the microservice with the Combiner service to combine JSON files into a single CSV or XLSX file.

http://origin/combiner/

FDNS Rules Microservice

This is the microservice with the Business Rules Engine for ingesting and validating JSON files.

http://origin/rules/
